◆ csp_bin_sem_create()

int csp_bin_sem_create ( csp_bin_sem_handle_t * sem)

Create/initialize a binary semaphore.

The semaphore is created in state unlocked (value 1).

On platforms supporting max values, the semaphore is created with a max value of 1, hence the naming binary.

Parameters
[in]semsemaphore
Returns
#CSP_SEMAPHORE_OK on success, otherwise #CSP_SEMAPHORE_ERROR
