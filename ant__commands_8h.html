writes the cumulative time (in 50ms increments) that the deployment system has been active (for a specified antenna) in a response buffer.

Parameters

responsea 2 byte buffer where the cumulative deployment time (in 50ms increments) will be written. divide the response by 20 to get seconds.
Returns
0 when the antenna deployment system has received the command, >0 otherwise

data written to the result buffer is only valid if 0 was returned. One should check this before using the result.
