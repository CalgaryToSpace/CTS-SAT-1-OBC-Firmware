void TIM_get_timestamp_string ( char * log_str,
size_t max_len )

Returns a computer-friendly timestamp string.

Parameters
log_str- Pointer to buffer that stores the log string
max_len- Maximum length of log_str buffer @detail The string identifies the timestamp of the last time synchronization, the synchronization source (N - none; G - GNSS/GPS; T - telecommand), and the time passed in ms since the last synchronization. Added together, the two numbers represent the current timestamp in ms. Example: "1719169299720+0000042000_N"

◆ TIM_get_timestamp_string_datetime()

void TIM_get_timestamp_string_datetime ( char * log_str,
size_t max_len )

Returns a human-friendly timestamp string.

Parameters
log_str- Pointer to buffer that stores the log string
max_len- Maximum length of log_str buffer @detail The string identifies the current Unix time, and number of milliseconds since the time was last synchronized to a reference source. String format: yyyymmddTHHMMSS.sss_X_rrr...

where

yyyy: year mm: month (01 to 12) dd: day of month (01 to 31) HH: hour (00 to 24) MM: minute (00 to 59) SS: second (00 to 60) sss: millisecond X: last synchronization source, one of N (none), G (GNSS/GPS), T (telecommand) rrr... milliseconds since last time synchronization Example output: 20240623T180132.142_T_420204
