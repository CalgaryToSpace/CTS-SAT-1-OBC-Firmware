EPS_CHANNEL_enum_t EPS_channel_from_str ( const char channel_name[])

Converts an EPS channel name to a channel enum.

Parameters
channel_nameA lowercase c-string of the channel name (e.g., "mpi"), or a number representing the channel number (e.g., "1" or "16"). Valid string values: "vbatt_stack", "stack_5v", "stack_3v3", "camera", "uhf_antenna_deploy", "lora_module", "mpi_5v", "mpi_12v", "boom".
Returns
The corresponding enum value. Returns EPS_CHANNEL_UNKNOWN if no match is found.
