◆ GPS_set_uart_interrupt_state()

void GPS_set_uart_interrupt_state ( uint8_t new_enabled)

Sets the UART interrupt state (enabled/disabled)

Parameters
new_enabled1: enable interrupt; 0: disable interrupt
Note
This function must be called very carefully. This type of GPS is known to, in the wrong mode, spam null bytes, which can lock up the entire system. Thus, the interrupt is disabled by default, and must be enabled explicitly by the GPS telecommands.
